# Team Pulse

팀원이 지금 어떤 파일을 작업 중인지 VS Code 사이드바에서 실시간으로 확인할 수 있는 확장 프로그램이에요.

## 주요 기능

- **실시간 활동 공유** — 팀원이 현재 열고 있는 파일을 즉시 확인
- **방 코드 시스템** — 8자리 초대 코드로 팀 방 생성 및 참가
- **GitHub 로그인** — GitHub OAuth로 인증, 특정 레포 collaborator만 입장 제한 가능
- **수정 파일 목록** — git 기준으로 팀원이 변경 중인 파일 목록 확인
- **상태 표시** — 온라인 / 자리비움 / 오프라인 상태를 실시간으로 표시
- **자동 재연결** — VS Code 재시작 시 자동으로 방에 재접속
- **프로필 사진** — GitHub 프로필 이미지 자동 표시
- **30일 세션** — 토큰과 방은 30일 후 자동 만료

## 시작하기

1. VS Code 확장 탭에서 `Team Pulse IMJEMIN` 검색 후 설치
2. 사이드바의 **Team Pulse** 아이콘 클릭
3. **GitHub로 로그인**
4. **새 방 만들기** (GitHub 레포 연동으로 입장 제한 가능) 또는 **초대 코드로 참가**
5. 초대 코드를 팀원에게 공유하면 끝!

## 커맨드

| 커맨드 | 설명 |
|---|---|
| `Team Pulse: Connect` | 방에 연결 |
| `Team Pulse: Disconnect` | 현재 방에서 연결 끊기 |
| `Team Pulse: GitHub 다시 로그인` | GitHub 재인증 |
| `Team Pulse: GitHub 로그아웃` | 로그아웃 및 저장된 데이터 초기화 |
| `Team Pulse: 방 코드 초기화` | 저장된 방 코드 초기화 |

## 설정

| 설정 | 기본값 | 설명 |
|---|---|---|
| `teamPulse.serverUrl` | `wss://ws.imjemin.co.kr` | WebSocket 서버 URL |
| `teamPulse.autoConnect` | `true` | VS Code 시작 시 자동 연결 |

팀원과 공유되는 파일 경로는 워크스페이스 기준 **상대 경로**만 전송돼요.
